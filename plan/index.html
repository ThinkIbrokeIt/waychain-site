⚓ Launch Plan

All 5 phases complete · Chain live at waychain.org

Status: ✅ Complete

Chain is live at waychain.org. 80,000+ blocks at 1/sec, 3 validators, BoltDB persistence.

Phase 0: Foundation ✅

  • BoltDB persistence — accounts, blocks, node, txs buckets
  • Tx storage with hash index — O(1) lookup
  • Transaction lifecycle — sendRawTransaction, getByHash, getReceipt
  • Binary wire format serialization
  • Deploy gate — 3-layer enforcement (RPC, block, EVM)
  • P2P networking — TCP gob, 6 message types, mesh topology
  • 3-validator set with sqrt-weighted proposer lottery
  • Systemd service, graceful shutdown, health endpoint

Phase 1: Safety ✅

  • Dox_Dev badges — L1/L2/L3, gated contract deployment
  • Deploy gate enforcement at all 3 layers
  • Badge checker UI at /badge
  • Genesis curators: alice (L3), bob (L2), treasury (L3)

Phase 2: Infrastructure ✅

  • waychain.org via Vercel with HTTPS
  • Cloudflare Tunnel for RPC (api.waychain.org)
  • WebSocket support on same port
  • Rate limiting — 100 req/sec/IP
  • Token infrastructure — 1WAY, 2WAY specs, DEX contracts

Phase 3: Interfaces ✅

  • Block Explorer — latest blocks, search, detail views
  • Dashboard — real-time WS push, stats, lookup
  • Badge UI — check Dox_Dev level, descriptions

Phase 4: Real-Time ✅

  • eth_subscribe: newHeads, newPendingTransactions
  • SubscriptionManager with auto-cleanup
  • Dashboard WebSocket integration with HTTP fallback

Phase 5: Hardening ✅

  • 3 validators producing blocks with rotating proposers
  • Rate limiter: token bucket, 100/sec/IP, auto-refill
  • Structured logging: slog JSON format
