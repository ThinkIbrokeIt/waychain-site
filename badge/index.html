How It Works

The Lantern (Level I): Proof you are a unique human. Verified through a curator or existing Level 3 holder. Grants access to Class A contracts, oracle monitoring, and basic governance.

The Compass (Level II): Verified builder. Requires Level 1 + bonding period. Grants access to Class B contracts, validator seats, and professional oracle badges.

The Lighthouse (Level III): Elected curator. Community vote every 90 days. Can issue badges, manage deployment classes, and access emergency controls.

Identity is encrypted. Disclosure only on verified legal request with on-chain approval via community vote.
