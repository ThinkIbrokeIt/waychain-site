Quick Start

// Connect to WayChain
client := waychain.Connect("https://api.waychain.org")

// Check Dox_Dev badge level
level := client.Precompile("0x13").Call("checkLevel", deployerAddr)
if level < 2 {
  return "Insufficient Dox_Dev level"
}

// Deploy contract (badge verified automatically)
tx := client.Deploy(contractCode, deployerAddr)

Key Differences

WayChain uses SHA256, not keccak256

All function selectors, storage keys, and transaction hashes use SHA256. Standard EVM tooling uses keccak256 — adapt your ABI selectors.

FeatureWayChainStandard EVM
Hash functionSHA256keccak256
SignaturesEd25519ECDSA (secp256k1)
Tx formatBinary (not RLP)RLP-encoded
FeesFixed (0.001/0.005/0.01 WAY)Auction-based
Precompiles21 precompiles (0x0C-0x25)0x01-0x0A

# Current block

# Check Dox_Dev level
curl -X POST https://api.waychain.org   -H "Content-Type: application/json"   -d '{"jsonrpc":"2.0","method":"way_getDoxLevel","params":["0xYourAddress"],"id":1}'
