Quick Start

// Connect to WayChain
client := waychain.Connect("https://api.waychain.org")

// Check Dox_Dev badge level
level := client.Precompile("0x13").Call("checkLevel", deployerAddr)
if level < 2 {
  return "Insufficient Dox_Dev level"
}

// Deploy contract (badge verified automatically)
tx := client.Deploy(contractCode, deployerAddr)

Key Differences from Standard EVM

WayChain uses SHA256, not keccak256

All function selectors, storage keys, and transaction hashes use SHA256. Standard EVM tooling uses keccak256 — adapt your ABI selectors.

Dox_Dev Identity Requirements

  • Level 1 (Lantern) — Verified human. Can hold tokens, receive/from transfers.
  • Level 2 (Compass) — Can deploy contracts. SHA256 selector validation.
  • Level 3 (Lighthouse) — Can verify others. Curator privileges.

Deploy Gate (3 Layers)

  1. RPC Layer — HTTP check before transaction enters pool.
  2. Block Production — Proposer verifies all transactions.
  3. EVM Opcode — Runtime enforcement in every contract call.
