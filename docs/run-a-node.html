Run a WayChain Node

WayChain is built in Go. A full node runs the daemon with embedded EVM, P2P networking, and JSON-RPC server. Minimum requirements are modest — a 2-core VPS with 4GB RAM and 10GB SSD is sufficient.

What a Node Does

The WayChain daemon runs all four layers: BFT consensus (block production + finality), EVM execution (smart contracts + precompiles), P2P networking (block/tx gossip), and JSON-RPC server (HTTP + WebSocket). BoltDB handles persistent state.

Installation

1

Install Go

wget https://go.dev/dl/go1.26.4.linux-amd64.tar.gz
sudo rm -rf /usr/local/go
sudo tar -C /usr/local -xzf go1.26.4.linux-amd64.tar.gz
echo 'export PATH=$PATH:/usr/local/go/bin' >> ~/.bashrc
source ~/.bashrc
go version
2

Clone & Build

git clone https://github.com/ThinkIbrokeIt/waychain
cd waychain/consensus
go build .
# Produces 'waychain-consensus' binary (or 'waychain' via go build -o waychain)
3

Start the Daemon

# Development mode (ephemeral state)
WAYCHAIN_DEVNET=1 ./waychain-consensus

# Production mode (persistent state at ~/.waychain/chain.db)
./waychain-consensus

The daemon starts the RPC server on :9545 and P2P on :9100. Blocks begin producing immediately at 1 per second.

4

Verify It's Running

# Check chain ID
curl -s -X POST http://localhost:9545 \
  -H 'Content-Type: application/json' \
  -d '{"jsonrpc":"2.0","method":"eth_chainId","id":1}'

# Should return: {"jsonrpc":"2.0","result":"0x2718","id":1}

# Check latest block
curl -s -X POST http://localhost:9545 \
  -H 'Content-Type: application/json' \
  -d '{"jsonrpc":"2.0","method":"eth_blockNumber","id":1}'

Systemd Service (Production)

For persistent operation, set up a systemd service:

sudo tee /etc/systemd/system/waychain.service << 'EOF'
[Unit]
Description=WayChain Node
After=network.target

[Service]
Type=simple
User=YOUR_USER
WorkingDirectory=/home/YOUR_USER/waychain/consensus
ExecStart=/home/YOUR_USER/waychain/consensus/waychain-consensus
Restart=always
RestartSec=5
LimitNOFILE=65536

[Install]
WantedBy=multi-user.target
EOF

sudo systemctl daemon-reload
sudo systemctl enable --now waychain
sudo journalctl -u waychain -f --no-pager

Joining the Network

Current Network Status

WayChain currently runs 3 validators on the devnet. P2P peer discovery is active — new nodes automatically connect to the mesh. The validator set is expandable to 200 seats via governance.

To join as a validator, you need:

  • Dox_Dev Level 2+ — verified human identity (badges available via curator application)
  • 32,000+ WAY — validator stake requirement
  • A running full node — steps 1-4 above
  • Open P2P port (9100) on your firewall

To run as an attester (oracle):

  • 5,000+ WAY — oracle bond
  • Dox_Dev Level 2+ — verified professional
